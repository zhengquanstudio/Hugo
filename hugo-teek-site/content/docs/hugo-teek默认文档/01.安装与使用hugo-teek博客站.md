## 🎉 项目简介

**Hugo-Teek** 是一款基于 VitePress Teek 主题移植的 Hugo 静态网站主题，专为技术博客和文档站点设计。它保留了 VitePress Teek 的精美设计，同时利用 Hugo 的极速构建能力，为技术博客、文档库、知识管理系统提供完整的解决方案。

### ✨ 核心特性

::: tip 🎨 设计特性
- 精美界面：VitePress Teek 风格设计
- 响应式布局：完美适配桌面/平板/移动端
- 暗色模式：自动切换浅色/暗色主题
- 三栏布局：侧边栏 + 内容 + 目录
:::

::: info ⚡ 性能特性
- 极速构建：基于 Hugo 引擎
- 本地搜索：FlexSearch 离线全文搜索
- 懒加载：图片按需加载
- 热重载：开发时实时预览
:::

::: tip 🛠️ 功能特性
- 评论系统：Twikoo 评论支持
- 数据统计：百度/Umami/不蒜子
- 代码高亮：Monokai 风格 + 复制按钮
- VitePress 语法：自动转换容器语法
:::

## 🚀 安装步骤

### 1️⃣ 克隆项目

#### 🔗 HTTPS 方式

::: code-group

```bash [步骤 1 - 克隆➡️]
git clone https://cnb.cool/yuwen-gueen/hugo-teeker-theme.git
```

```bash [步骤 2 - 进入目录🎉]
cd hugo-teeker-theme
```

:::

#### 🔐 SSH 方式

::: code-group

```bash [步骤 1 - 克隆➡️]
git clone git@cnb.cool:yuwen-gueen/hugo-teeker-theme.git
```

```bash [步骤 2 - 进入目录🎉]
cd hugo-teeker-theme
```

:::

### 2️⃣ 安装构建工具

::: code-group

```bash [Linux/Mac🔧]
make install-tools
```

```batch [Windows🔧]
# 使用 PowerShell 执行
.\bin\windows\check-and-install.bat
```

```bash [Linux/Mac 启动后台配置中心🎛️]
sh bin/linux/start-admin.sh
```

```bash [Linux/Mac 后台配置中心说明🎛️]
执行命令启动后台配置中心
启动后访问: http://localhost:5173/admin
用于图形化配置博客各项参数
```

:::

### 3️⃣ 启动开发服务器

::: code-group

```bash [Linux/Mac 启动指令🚀]
make dev
```

```bash [Linux/Mac 启动说明🚀]
包含搜索功能、配置中心、内容监听
访问: http://localhost:8090
```

```bash [Linux/Mac 快速启动✨]
make quick
```

```bash [Linux/Mac 快速启动说明✨]
跳过预生成和搜索索引，快速预览
访问: http://localhost:8090
```

```batch [Windows 启动指令🚀]
# 方式1：双击运行（推荐）
bin\windows\start.bat

# 方式2：PowerShell 命令行
cd bin\windows
.\start.bat
```

```batch [Windows 启动说明🚀]
到项目路径 bin\windows\ 目录下
右键以管理员身份运行 start.bat
包含搜索功能、内容监听
访问: http://localhost:9090
需要重启项目时也可双击管理员运行
```

```batch [Windows 启动后台配置中心🎛️]
# 方式1：双击运行（推荐）
bin\windows\start-admin.bat

# 方式2：PowerShell 命令行
cd bin\windows
.\start-admin.bat
```

```batch [Windows 后台配置中心说明🎛️]
启动管理后台（前端 + 后端）
前端访问: http://localhost:6173/admin
后端 API: http://localhost:9888
用于图形化配置博客各项参数
```

## ⚙️ 配置文件说明

项目采用分层配置结构，便于管理和切换主题：

::: code-group

```toml [核心配置📄]
# hugo-teek-site/config/_default/hugo.toml
baseURL = "https://wiki.xxdevops.cn/" # 站点URL
title = "你的站点名称"
theme = "hugo-teek"
languageCode = "zh-CN"
```

```toml [主题配置📄]
# hugo-teek-site/config/themes/hugo-teek/params.toml
[search]
    enabled = true
    provider = "flexsearch"

[heroBg]
    enabled = true
    autoPlay = true
```

```toml [插件配置📄]
# hugo-teek-site/config/teek-plugins/site.toml
description = "站点描述"
keywords = ["博客", "技术", "Hugo"]
siteStartDate = "2025-01-01"
```

:::

### 配置合并机制

项目使用智能配置合并系统，执行 `make dev` 时会自动：

1. 合并 `teek-plugins/` 和 `themes/{theme}/` 下的配置
2. 生成统一的 `config/_default/params.toml`
3. 根据主题配置自动启用/禁用功能

## 🎨 主题切换

项目支持多主题切换，目前内置两款主题：

::: code-group

```toml [hugo-teek 主题🎨]
# hugo-teek-site/config/_default/hugo.toml
theme = "hugo-teek"
# 功能丰富，适合文档库和博客
```

```toml [hugo-simple 主题🎨]
# hugo-teek-site/config/_default/hugo.toml
theme = "hugo-simple"
# 简洁轻量，适合个人博客
```

:::

切换主题后执行：

```bash
make dev
# 系统会自动检测主题并加载对应配置
```

## 🔍 搜索功能

搜索功能基于 FlexSearch，提供极速离线全文搜索体验。

### 启用搜索

::: code-group

```toml [启用搜索✅]
# config/themes/hugo-teek/params.toml
[search]
    enabled = true
    provider = "flexsearch"
```

```toml [禁用搜索❌]
# config/themes/hugo-teek/params.toml
[search]
    enabled = false
```

:::

### 生成搜索索引

```bash
# 开发模式会自动生成
make dev

# 手动生成搜索索引
make gen-search-index
```

## 💬 评论系统

内置 Twikoo 评论系统，支持多种部署方式。

### 配置评论

```toml
# config/teek-plugins/services.toml
[comment]
    enabled = true
    type = "twikoo"
    envId = "你的 Twikoo 环境 ID"
```

## 🛠️ 常用命令

### 开发命令

| 命令 | 说明 |
|------|------|
| `make dev` | 启动完整开发环境 |

## 🔧 故障排除

### 常见问题

::: code-group

```markdown [端口被占用❌]
# 错误：端口 9090 已被占用
# 解决：make dev 会自动清理端口
# 或手动指定端口：
make dev FRONTEND_PORT=8080
```

```markdown [搜索不工作❌]
# 检查搜索配置是否启用
# 重新生成搜索索引：
make gen-search-index
```

```markdown [样式丢失❌]
# 清理并重新构建：
make clean
make dev
```

```markdown [无法启动start.bat❌]
# 错误：The term 'check-and-install.bat' is not recognized
# 原因：PowerShell 默认不从当前目录加载命令
#
# 完整解决步骤：
# 1. 切换到正确目录
#    cd 盘符\hugo-teeker-theme\bin\windows
#
# 2. 运行安装脚本（注意加 .\）
#    .\check-and-install.bat
#
# 3. 等待下载完成后，运行启动脚本
#    .\start.bat
```
