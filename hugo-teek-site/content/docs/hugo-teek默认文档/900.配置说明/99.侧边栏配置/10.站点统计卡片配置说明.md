## 📊 站点统计卡片概述

站点统计卡片展示站点的基本统计信息，包括文章数、分类数、标签数等，让访客快速了解站点规模，也展示了博主的创作成果。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 数字展示：醒目的统计数字
- 图标装饰：每个统计项配有图标
- 布局整齐：网格或列表布局
- 实时更新：数据自动统计
:::

::: info ⚙️ 功能特性
- 自动统计：自动计算站点数据
- 无需维护：数据实时更新
- 独立开关：可随时启用/禁用
- 多项统计：文章、分类、标签等
:::

## 📝 开关配置

### 启用/禁用站点统计卡片

::: code-group

```toml [启用站点统计✅]
[params.homeSidebar.siteInfo]
    enabled = true   # ◀️ 显示站点统计卡片
```

```toml [禁用站点统计❌]
[params.homeSidebar.siteInfo]
    enabled = false  # ◀️ 隐藏站点统计卡片
```

:::

## 📈 统计项说明

### 自动统计内容

站点统计卡片自动统计以下信息：

::: code-group

```yaml [统计项📊]
📄 文章总数：站点所有已发布文章数量
📂 分类数量：站点所有分类数量
🏷️ 标签数量：站点所有标签数量
📅 运行天数：站点自创建以来的运行天数
```

```yaml [显示样式📋]
┌─────────────────────────────┐
│ 📊 站点统计                   │
├─────────────────────────────┤
│                             │
│   📄 文章      128 篇        │
│   📂 分类       12 个        │
│   🏷️ 标签       56 个        │
│   📅 运行      365 天        │
│                             │
└─────────────────────────────┘
```

:::

### 统计数据来源

::: info 🤖 数据来源
站点统计卡片数据完全自动统计：

1️⃣ **文章数**：统计 `content/` 目录下所有 `draft: false` 的文章
2️⃣ **分类数**：统计所有文章中 `categories` 的唯一值数量
3️⃣ **标签数**：统计所有文章中 `tags` 的唯一值数量
4️⃣ **运行天数**：从第一篇发布的文章日期计算至今
:::

## 🔧 常见问题

### 统计数据不准确

::: code-group

```markdown [问题：数据不准确❌]
原因1：包含草稿文章
解决：
1️⃣ 检查文章 draft 字段
2️⃣ 确保草稿文章设置为 draft: true
3️⃣ 重新构建站点

原因2：缓存问题
解决：
1️⃣ 执行 make clean 清理缓存
2️⃣ 重新执行 make dev

原因3：统计范围错误
解决：
1️⃣ 检查 content/ 目录结构
2️⃣ 确保文章在正确的目录下
```

:::

### 运行天数计算错误

::: code-group

```markdown [问题：运行天数错误❌]
原因：最早文章日期设置错误
解决：
1️⃣ 检查最早发布的文章日期
2️⃣ 确保第一篇发布的文章 date 字段正确
3️⃣ 运行天数从最早文章日期开始计算
```

:::
