## ⭐ 精选文章卡片概述

精选文章卡片展示被标记为【精选】的文章列表，用于突出显示优质内容，帮助访客快速发现站点的精华文章。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 精选标记：突出显示推荐文章
- 缩略图展示：显示文章封面图
- 标题列表：简洁的文章标题列表
- 快速跳转：点击直达文章详情
:::

::: info ⚙️ 功能特性
- 自动收集：自动收集 featured: true 的文章
- 排序规则：按发布时间倒序排列
- 独立开关：可随时启用/禁用
- 数量限制：自动限制显示数量
:::

## 📝 开关配置

### 启用/禁用精选文章卡片

::: code-group

```toml [启用精选文章✅]
[params.homeSidebar.featuredPosts]
    enabled = true   # ◀️ 显示精选文章卡片
```

```toml [禁用精选文章❌]
[params.homeSidebar.featuredPosts]
    enabled = false  # ◀️ 隐藏精选文章卡片
```

:::

## 📝 文章标记方法

### 设置文章为精选

在文章 Front Matter 中设置 `featured: true`：

::: code-group

```yaml [精选文章标记⭐]
---
title: "文章标题"
date: "2026-03-02T10:00:00+08:00"
featured: true    # ◀️ 标记为精选文章
---

文章内容...
```

```yaml [普通文章📄]
---
title: "普通文章"
date: "2026-03-02T10:00:00+08:00"
featured: false   # ◀️ 不标记为精选（默认）
---

文章内容...
```

:::

### 批量标记精选文章

::: code-group

```yaml [多篇精选文章📋]
# 文章1：精选
---
title: "Hugo-Teek 主题安装教程"
featured: true
---

# 文章2：精选
---
title: "Markdown 语法大全"
featured: true
---

# 文章3：普通
---
title: "日常随笔"
featured: false
---
```

:::

## 🎨 显示规则

### 自动收集机制

::: info 🤖 收集规则
精选文章卡片自动收集并显示文章：

1️⃣ **筛选条件**：`featured: true` 的文章
2️⃣ **排序规则**：按 `date` 字段倒序（最新在前）
3️⃣ **数量限制**：自动限制显示数量（通常 5-10 篇）
4️⃣ **更新时机**：每次构建时重新收集
:::

## 📋 完整配置示例

### 基础配置

::: code-group

```toml [启用精选文章📋]
# homepage.toml
[params.homeSidebar.featuredPosts]
    enabled = true
```

```yaml [标记精选文章📋]
# content/posts/my-post.md
---
title: "Hugo-Teek 主题安装教程"
date: "2026-03-02T10:00:00+08:00"
featured: true
---
```

:::

### 配置位置示例

::: code-group

```toml [完整 homepage.toml 示例📋]
# ========================================
# 首页配置 (Homepage Configuration)
# ========================================
[params.homeSidebar]

  # ========================================
  # 精选文章卡片 (Featured Posts Card)
  # ========================================
  # 显示标记为精选的文章
  # 数据来源：文章 Front Matter 中设置 featured: true
  [params.homeSidebar.featuredPosts]
    enabled = true  # ◀️ 开关：启用/禁用精选文章卡片
```

:::

## 🔧 常见问题

### 精选文章不显示

::: code-group

```markdown [问题：精选文章不显示❌]
原因1：开关未启用
解决：检查 homepage.toml 中 enabled = true

原因2：无精选文章
解决：
1️⃣ 检查文章 Front Matter 中是否设置 featured: true
2️⃣ 确保文章已发布（draft: false）
3️⃣ 执行 make dev 重新构建

原因3：文章未发布
解决：
1️⃣ 检查文章 draft 字段
2️⃣ 确保 draft: false 或删除该字段
```

:::

### 文章排序异常

::: code-group

```markdown [问题：排序不正确❌]
原因：日期格式错误或时区问题
解决：
1️⃣ 使用标准 ISO 8601 格式：YYYY-MM-DDTHH:MM:SS+08:00
2️⃣ 确保所有精选文章都有正确的 date 字段
3️⃣ 重新构建站点
```

:::
