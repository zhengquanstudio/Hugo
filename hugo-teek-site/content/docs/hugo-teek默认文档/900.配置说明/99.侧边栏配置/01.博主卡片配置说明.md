## 👤 博主卡片概述

博主卡片是 Hugo-Teek 主题首页侧边栏的核心组件，始终显示在侧边栏最顶部（无开关控制）。它展示博主的个人信息、头像、状态表情和社交链接，是站点的【个人名片】。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 圆形旋转头像：支持多种形状和动画效果
- 动态背景：可设置个性化背景图片
- 状态表情：右下角显示当前状态图标
- 社交链接：支持多个社交平台链接
:::

::: info ⚙️ 功能特性
- 响应式布局：自适应不同屏幕尺寸
- 懒加载优化：头像和背景图片按需加载
- 外链跳转：社交链接支持新窗口打开
- 图标支持：内置多种社交图标
:::

博主卡片配置位于主题专属配置文件中：

::: code-group

```toml [主题配置文件📄]
# hugo-teek-site/config/themes/hugo-teek/params.toml
# 博主卡片配置位于 [blogger] 章节
```

```toml [配置章节定位📝]
# ========================================
# 博主卡片 (Blogger Card)
# ========================================
[blogger]
    # 博主卡片所有配置项...
```

:::

## 📝 基础配置项

### 头像配置

博主头像支持多种形状和样式：

::: code-group

```toml [头像基础配置📝]
[blogger]
    avatar = "https://img.xxdevops.cn/blog/avatar/yuwen_avatar.avif?w=150&h=150&fit=crop&fm=webp&q=80"
    name = "余温Gueen"
    slogan = "明心静性，爱自己"
```

```toml [配置说明📝]
[blogger]
    avatar = "图片URL"     # ◀️ 头像图片地址，建议使用 CDN
    name = "博主名称"       # ◀️ 显示名称
    slogan = "个性签名"     # ◀️ 个性签名/标语
```

:::

### 头像形状配置

通过 `shape` 字段设置头像形状：

::: code-group

```toml [头像形状配置🎨]
[blogger]
    shape = "circle-rotate"   # ◀️ 圆形旋转效果
```

```toml [可选形状值🎨]
[blogger]
    shape = "circle"          # ◀️ 普通圆形
    shape = "circle-rotate"   # ◀️ 圆形带旋转动画
    shape = "square"          # ◀️ 正方形
    shape = "rounded"         # ◀️ 圆角矩形
```

:::

## 🖼️ 背景图片配置

### 卡片背景

博主卡片支持设置个性化背景图片：

::: code-group

```toml [背景图片配置🖼️]
[blogger]
    backgroundImage = "https://img.xxdevops.cn/blog/wallpaper/bg14.avif"
```

```toml [背景配置说明📝]
[blogger]
    # ◀️ 背景图片 URL，留空则使用纯色背景
    backgroundImage = "https://example.com/bg.jpg"
    
    # ◀️ 不设置或设置为空字符串使用默认纯色背景
    backgroundImage = ""
```

:::

### 圆形背景图（装饰效果）

::: code-group

```toml [圆形背景配置⭕]
[blogger]
    circleBgImg = "https://img.xxdevops.cn/blog/avatar/yuwen_avatar.avif?w=300&h=300&fit=crop&fm=webp&q=80"
    circleBgMask = false
```

```toml [圆形背景说明📝]
[blogger]
    # ◀️ 圆形装饰背景图，通常使用放大版头像
    circleBgImg = "图片URL"
    
    # ◀️ 是否启用遮罩效果，true 时添加渐变遮罩
    circleBgMask = false
```

:::

## 😊 状态表情配置

### 右下角状态图标

博主卡片右下角可以显示状态表情图标：

::: code-group

```toml [状态表情配置😊]
[blogger.status]
    icon = "😪"           # ◀️ 状态表情符号
    size = 18.0           # ◀️ 图标大小（像素）
    title = "困"          # ◀️ 鼠标悬停提示文字
```

```toml [常用状态示例😊]
[blogger.status]
    icon = "😊"           # ◀️ 开心
    title = "心情不错"

[blogger.status]
    icon = "💻"           # ◀️ 工作中
    title = "正在coding"

[blogger.status]
    icon = "☕"           # ◀️ 休息中
    title = "喝咖啡中"

[blogger.status]
    icon = "🎮"           # ◀️ 游戏中
    title = "正在游戏"
```

## 🔗 社交链接配置

### 添加社交链接

社交链接显示在博主名称下方、个性签名上方：

::: code-group

```toml [社交链接配置🔗]
# GitHub
[[blogger.social]]
    name = "GitHub"
    icon = "github"
    url = "https://github.com/JSXGQ"

# CNB
[[blogger.social]]
    name = "CNB"
    icon = "cnb"
    url = "https://cnb.cool/u/yuwenGueen"

# QQ
[[blogger.social]]
    name = "QQ"
    icon = "qq"
    url = "tencent://message/?uin=你的QQ号"

# 微信
[[blogger.social]]
    name = "微信"
    icon = "weichat"
    url = "#"
```

```toml [配置字段说明📝]
[[blogger.social]]
    name = "显示名称"      # ◀️ 社交平台的显示名称
    icon = "图标名称"      # ◀️ 图标文件名（不含扩展名）
    url = "链接地址"       # ◀️ 点击跳转的链接
```

:::

## 🎨 图片优化建议

### 头像图片规格

::: tip 📐 推荐尺寸
- **头像尺寸**: 150×150 像素（显示尺寸）
- **实际尺寸**: 300×300 像素（2x 倍图，适配高分屏）
- **图片格式**: WebP（推荐）或 JPG
- **压缩质量**: 80%
- **文件大小**: 建议控制在 50KB 以内
:::

### CDN 参数示例

::: code-group

```toml [Imgix CDN 格式🌐]
# Imgix 格式
avatar = "https://example.imgix.net/avatar.jpg?w=150&h=150&fit=crop&fm=webp&q=80"
```

```toml [七牛云 CDN 格式🌐]
# 七牛云格式
avatar = "https://example.qiniudn.com/avatar.jpg?imageView2/1/w/150/h/150/q/80"
```

```toml [又拍云 CDN 格式🌐]
# 又拍云格式
avatar = "https://example.b0.upaiyun.com/avatar.jpg!/fw/150/fh/150/quality/80"
```

## 🔧 常见问题

### 头像不显示

::: code-group

```markdown [问题：头像不显示❌]
原因：图片链接失效或跨域问题
解决：
1️⃣ 检查图片 URL 是否可访问
2️⃣ 使用 HTTPS 协议
3️⃣ 检查图片是否允许跨域访问
4️⃣ 尝试使用 CDN 托管图片
```

```markdown [问题：社交图标不显示❌]
原因：icon 名称错误或图标文件不存在
解决：
1️⃣ 确认使用内置图标名称
2️⃣ 检查图标文件是否存在于主题目录
3️⃣ 自定义图标需放在 assets/icons/ 目录
```

:::

### 配置生效检查清单

