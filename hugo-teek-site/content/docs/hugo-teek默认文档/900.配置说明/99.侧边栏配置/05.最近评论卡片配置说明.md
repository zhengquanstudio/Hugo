## 💬 最近评论卡片概述

最近评论卡片展示站点最新的评论内容，增强站点互动性，让访客快速了解最新的讨论热点。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 评论预览：显示评论内容和评论者
- 头像展示：显示评论者头像
- 文章关联：显示评论所属文章
- 时间显示：显示评论发布时间
:::

::: info ⚙️ 功能特性
- 实时获取：从评论系统实时获取数据
- 分页浏览：支持翻页查看更多评论
- 数量控制：可配置显示数量
- 依赖评论系统：需要启用评论功能
:::

## 📝 开关配置

### 启用/禁用最近评论卡片

::: code-group

```toml [启用最近评论✅]
[params.homeSidebar.recentComments]
    enabled = true    # ◀️ 显示最近评论卡片
    pageSize = 5      # ◀️ 每页显示评论数量
    totalSize = 50    # ◀️ 总共获取评论数量
```

```toml [禁用最近评论❌]
[params.homeSidebar.recentComments]
    enabled = false   # ◀️ 隐藏最近评论卡片
```

:::

## ⚙️ 配置参数

### 评论数量配置

::: code-group

```toml [数量配置📝]

    enabled = true
    pageSize = 5      # ◀️ 每页显示评论数量
    totalSize = 50    # ◀️ 总共获取评论数量
```

```toml [参数说明📝]
[params.homeSidebar.recentComments]
    pageSize = 5      # ◀️ 每页显示条数，建议 3-10
```

## 🔗 评论系统依赖

### 前置条件

最近评论卡片依赖评论系统，需要先启用评论功能：

::: code-group

```toml [Twikoo 评论配置✅]
# services.toml
[comment]
    enabled = true           # ◀️ 启用评论系统
    type = "twikoo"          # ◀️ 评论系统类型
    envId = "你的环境ID"      # ◀️ Twikoo 环境 ID
```

```toml [评论开关检查📝]
# 确保以下配置为 true
[comment]
    enabled = true
```

:::

## 🔧 常见问题

### 评论不显示

::: code-group

```markdown [问题：评论不显示❌]
原因1：评论系统未启用
解决：
1️⃣ 检查 services.toml 中 comment.enabled = true
2️⃣ 确保已配置正确的 envId

原因2：评论卡片开关未启用
解决：检查 homepage.toml 中 recentComments.enabled = true

原因3：无评论数据
解决：
1️⃣ 确保站点已有评论
2️⃣ 检查评论系统是否正常工作
3️⃣ 检查网络连接是否正常
```

:::

### 评论加载失败

::: code-group

```markdown [问题：评论加载失败❌]
原因：评论系统配置错误或网络问题
解决：
1️⃣ 检查 envId 是否正确
2️⃣ 检查评论系统服务是否正常
3️⃣ 检查浏览器控制台是否有报错
4️⃣ 确认评论系统域名是否可访问
```

:::
