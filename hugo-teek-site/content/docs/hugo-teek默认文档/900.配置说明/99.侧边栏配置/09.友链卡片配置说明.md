## 🔗 友链卡片概述

友链卡片展示站点的友情链接，支持分组显示和自动轮播，方便访客发现其他优质博客，也是博主之间交流互动的重要方式。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 分组显示：支持按分组显示友链
- 自动轮播：多组友链自动切换
- 头像展示：显示友站点头像/Logo
- 悬停信息：鼠标悬停显示站点描述
:::

::: info ⚙️ 功能特性
- 分组管理：支持多个友链分组
- 数量限制：限制显示友链数量
- 独立开关：可随时启用/禁用
- 数据驱动：通过 YAML 文件管理
:::

## 📝 开关配置

### 启用/禁用友链卡片

::: code-group

```toml [启用友链✅]
[params.homeSidebar.friendLinks]
    enabled = true   # ◀️ 显示友链卡片
```

```toml [禁用友链❌]
[params.homeSidebar.friendLinks]
    enabled = false  # ◀️ 隐藏友链卡片
```

:::

## ⚙️ 友链参数配置

### 友链显示参数

在 `params.toml` 中配置友链的显示参数：

::: code-group

```toml [友链参数📝]
[friendLink]
    limit = 5           # ◀️ 每组显示友链数量
    interval = 5000     # ◀️ 轮播间隔（毫秒）
    groups = ["Teek道友"]  # ◀️ 指定显示的分组
```

```toml [参数说明📝]
[friendLink]
    limit = 5           # ◀️ 每组最多显示多少个友链
    interval = 5000     # ◀️ 自动轮播间隔，5000ms = 5秒
    groups = []         # ◀️ 显示哪些分组，空数组表示全部
```

:::

### 分组配置

::: code-group

```toml [显示全部分组📝]
[friendLink]
    groups = []         # ◀️ 空数组表示显示所有分组
```

```toml [指定分组📝]
[friendLink]
    groups = ["Teek道友"]           # ◀️ 只显示指定分组
    groups = ["Teek道友", "VitePress"]  # ◀️ 显示多个分组
```

:::

### 友链独立页面配置

::: code-group

```toml [友链页面配置📝]
[friendPage]
    smallTitle = "与各位博主一起成长进步"
    bannerButtonGroup = true
```

```toml [配置说明📝]
[friendPage]
    smallTitle = "副标题文字"      # ◀️ 友链页面副标题
    bannerButtonGroup = true       # ◀️ 是否显示分组按钮
```

:::

## 📝 友链数据配置

### 友链数据文件

编辑 `data/friendLinks.yaml` 配置友链信息：

::: code-group

```yaml [友链数据格式🔗]
groups:
  - title: "Teek道友"           # ◀️ 分组名称
    links:
      - name: "站点名称"         # ◀️ 站点名称
        url: "https://example.com"  # ◀️ 站点链接
        avatar: "头像URL"        # ◀️ 站点头像
        description: "站点描述"   # ◀️ 站点描述
```

```yaml [完整示例📋]
groups:
  - title: "Teek道友"
    links:
      - name: "余温小站"
        url: "https://wiki.xxdevops.cn"
        avatar: "https://img.xxdevops.cn/blog/avatar/yuwen_avatar.avif"
        description: "明心静性，爱自己"
      
      - name: "白木の小屋"
        url: "https://gl.baimu.live"
        avatar: "https://gl.baimu.live/img/avatar.png"
        description: "白木的博客"
  
  - title: "技术博客"
    links:
      - name: "示例博客"
        url: "https://example.com"
        avatar: "https://example.com/avatar.png"
        description: "技术分享"
```

:::

### 友链字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `name` | ✅ | 站点名称 |
| `url` | ✅ | 站点链接 |
| `avatar` | ❌ | 站点头像/Logo |
| `description` | ❌ | 站点描述 |

## 🔧 常见问题

### 友链不显示

::: code-group

```markdown [问题：友链不显示❌]
原因1：开关未启用
解决：检查 homepage.toml 中 enabled = true

原因2：数据文件错误
解决：
1️⃣ 检查 data/friendLinks.yaml 是否存在
2️⃣ 检查 YAML 语法是否正确
3️⃣ 确保 groups 和 links 结构正确

原因3：分组配置错误
解决：
1️⃣ 检查 params.toml 中 groups 配置
2️⃣ 确保分组名称与数据文件中的一致
3️⃣ 或使用空数组显示所有分组
```

:::

### 头像不显示

::: code-group

```markdown [问题：头像不显示❌]
原因：头像链接失效或跨域
解决：
1️⃣ 检查 avatar URL 是否可访问
2️⃣ 使用 HTTPS 协议
3️⃣ 使用 CDN 托管头像图片
```

:::
