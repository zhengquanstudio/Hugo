## 📂 分类卡片概述

分类卡片展示站点的文章分类统计信息，以列表形式显示各个分类及其文章数量，帮助访客快速了解站点内容结构。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 分类列表：显示所有文章分类
- 数量统计：显示每个分类的文章数
- 快速跳转：点击分类进入分类页面
- 图标装饰：每个分类可配置图标
:::

::: info ⚙️ 功能特性
- 自动统计：自动统计各分类文章数量
- 数量排序：可按文章数排序
- 独立开关：可随时启用/禁用
- 响应式布局：自适应显示
:::

## 📝 开关配置

### 启用/禁用分类卡片

::: code-group

```toml [启用分类✅]
[params.homeSidebar.categories]
    enabled = true   # ◀️ 显示分类卡片
```

```toml [禁用分类❌]
[params.homeSidebar.categories]
    enabled = false  # ◀️ 隐藏分类卡片
```

:::

## ⚙️ 分类参数配置

### 分类显示参数

在 `params.toml` 中配置分类的显示参数：

::: code-group

```toml [分类参数📝]
[category]
    autoPage = false      # ◀️ 是否自动分页
    enabled = true        # ◀️ 是否启用分类功能
    limit = 8.0           # ◀️ 显示分类数量限制
    pageSpeed = 4000.0    # ◀️ 自动翻页速度（毫秒）
```

```toml [参数说明📝]
[category]
    autoPage = false      # ◀️ 自动分页开关，true 时自动轮播
    enabled = true        # ◀️ 分类功能总开关
    limit = 8.0           # ◀️ 最多显示多少个分类
    pageSpeed = 4000.0    # ◀️ 自动翻页间隔，4000ms = 4秒
```

:::

### 参数配置建议

::: code-group

```toml [分类较少时📝]
[category]
    autoPage = false      # ◀️ 分类少，不需要自动分页
    enabled = true
    limit = 10.0          # ◀️ 显示全部分类
    pageSpeed = 4000.0
```

```toml [分类较多时📝]
[category]
    autoPage = true       # ◀️ 分类多，启用自动轮播
    enabled = true
    limit = 8.0           # ◀️ 每页显示8个
    pageSpeed = 4000.0    # ◀️ 4秒切换一次
```

:::

## 📝 文章分类设置

### 在文章中设置分类

在文章 Front Matter 中使用 `categories` 字段：

::: code-group

```yaml [单分类📂]
---
title: "文章标题"
categories:
    - 技术
---
```

```yaml [多分类📂]
---
title: "文章标题"
categories:
    - 技术
    - 前端
    - 教程
---
```

:::

### 分类层级

Hugo 支持分类层级结构：

::: code-group

```yaml [层级分类📂]
---
title: "文章标题"
categories:
    - 技术/前端/React
---

# 或
---
title: "文章标题"
categories:
    - 技术
    - 技术/前端
    - 技术/前端/React
---
```

:::

## 🔧 常见问题

### 分类不显示

::: code-group

```markdown [问题：分类不显示❌]
原因1：开关未启用
解决：
1️⃣ 检查 homepage.toml 中 enabled = true
2️⃣ 检查 params.toml 中 category.enabled = true

原因2：无分类数据
解决：
1️⃣ 检查文章是否设置了 categories
2️⃣ 确保文章已发布（draft: false）
3️⃣ 执行 make dev 重新构建

原因3：limit 设置为 0
解决：检查 params.toml 中 category.limit 是否大于 0
```

:::

### 分类数量不准确

::: code-group

```markdown [问题：数量不准确❌]
原因：构建缓存问题
解决：
1️⃣ 执行 make clean 清理缓存
2️⃣ 重新执行 make dev
3️⃣ 检查文章分类设置是否正确
```

:::
