## 🏷️ 热门标签卡片概述

热门标签卡片以标签云形式展示站点的热门标签，标签大小代表使用频率，帮助访客快速了解站点内容主题，也方便发现相关文章。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 标签云展示：标签大小表示使用频率
- 颜色区分：不同标签有不同颜色
- 悬停效果：鼠标悬停高亮显示
- 快速跳转：点击标签查看相关文章
:::

::: info ⚙️ 功能特性
- 自动统计：自动统计标签使用频率
- 热门排序：按使用次数排序
- 独立开关：可随时启用/禁用
- 数量限制：限制显示标签数量
:::

# hugo-teek-site/config/themes/hugo-teek/homepage.toml
# 控制热门标签卡片的显示/隐藏

## 📝 开关配置

### 启用/禁用热门标签卡片

::: code-group

```toml [启用热门标签✅]
[params.homeSidebar.hotTags]
    enabled = true   # ◀️ 显示热门标签卡片
```

```toml [禁用热门标签❌]
[params.homeSidebar.hotTags]
    enabled = false  # ◀️ 隐藏热门标签卡片
```

:::

## 📝 文章标签设置

### 在文章中设置标签

在文章 Front Matter 中使用 `tags` 字段：

::: code-group

```yaml [单标签🏷️]
---
title: "文章标题"
tags:
    - Hugo
---
```

```yaml [多标签🏷️]
---
title: "文章标题"
tags:
    - Hugo
    - 博客
    - 教程
    - 静态网站
---
```

:::

### 标签最佳实践

::: tip 📝 标签使用建议
- **数量控制**：每篇文章建议 3-5 个标签
- **标签命名**：使用简洁、通用的词汇
- **避免重复**：不要创建语义重复的标签
- **统一规范**：建立标签命名规范，如统一小写
:::

## 🎨 标签云说明

### 标签大小规则

标签云根据标签使用频率显示不同大小：

::: code-group

```yaml [大小规则📊]
特大标签：使用 20+ 次
大标签：   使用 10-19 次
中标签：   使用 5-9 次
小标签：   使用 1-4 次
```

```yaml [显示样式📋]
┌─────────────────────────────┐
│ 🏷️ 热门标签                   │
├─────────────────────────────┤
│                             │
│   Hugo    博客    教程       │
│      前端      运维          │
│   笔记   工具    配置        │
│                             │
└─────────────────────────────┘
```

:::

### 标签颜色

::: info 🎨 颜色规则
标签颜色根据使用频率自动分配：
- **高频标签**：醒目的主题色
- **中频标签**：标准文字色
- **低频标签**：次要文字色
:::

## 📋 完整配置示例

### 基础配置

::: code-group

```toml [启用热门标签📋]
# homepage.toml
[params.homeSidebar.hotTags]
    enabled = true
```

```yaml [文章标签示例📋]
# content/posts/my-post.md
---
title: "Hugo-Teek 主题安装教程"
date: "2026-03-02T10:00:00+08:00"
tags:
    - Hugo
    - 博客
    - 教程
    - 主题
---
```

:::

### 配置位置示例

::: code-group

```toml [完整 homepage.toml 示例📋]
# ========================================
# 首页配置 (Homepage Configuration)
# ========================================
[params.homeSidebar]

  # ========================================
  # 热门标签卡片 (Hot Tags Card)
  # ========================================
  # 显示热门标签云
  [params.homeSidebar.hotTags]
    enabled = true  # ◀️ 开关：启用/禁用热门标签卡片
```

:::

## 🔧 常见问题

### 标签不显示

::: code-group

```markdown [问题：标签不显示❌]
原因1：开关未启用
解决：检查 homepage.toml 中 enabled = true

原因2：无标签数据
解决：
1️⃣ 检查文章是否设置了 tags
2️⃣ 确保文章已发布（draft: false）
3️⃣ 执行 make dev 重新构建

原因3：标签数量过少
解决：
1️⃣ 为更多文章添加标签
2️⃣ 确保标签使用频率达到显示阈值
```

:::

### 标签云样式异常

::: code-group

```markdown [问题：样式异常❌]
原因：CSS 冲突或加载失败
解决：
1️⃣ 检查浏览器控制台是否有 CSS 错误
2️⃣ 清除浏览器缓存
3️⃣ 检查主题文件是否完整
```

:::
