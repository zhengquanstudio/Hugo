## 📢 公告卡片概述

公告卡片用于在首页侧边栏显示站点公告信息，适合发布重要通知、更新日志、活动推广等内容。

### ✨ 卡片特性

::: tip 🎨 设计特性
- 醒目展示：位于侧边栏显眼位置
- 富文本支持：支持 Markdown 格式
- 图标装饰：可自定义公告图标
- 滚动效果：长内容自动滚动显示
:::

::: info ⚙️ 功能特性
- 独立开关：可随时启用/禁用
- 数据驱动：通过 YAML 文件管理内容
- 实时更新：修改后自动生效
:::

---

## 📁 配置文件路径

公告卡片涉及两个配置文件：

::: code-group

```toml [开关配置📄]
# hugo-teek-site/config/themes/hugo-teek/homepage.toml
# 控制公告卡片的显示/隐藏
```

```yaml [内容配置📄]
# hugo-teek-site/data/announcement.yaml
# 编辑公告的具体内容
```

:::

---

## 📝 开关配置

### 启用/禁用公告卡片

在 `homepage.toml` 中控制公告卡片的显示：

::: code-group

```toml [启用公告✅]
[params.homeSidebar.announcement]
    enabled = true   # ◀️ 显示公告卡片
```

```toml [禁用公告❌]
[params.homeSidebar.announcement]
    enabled = false  # ◀️ 隐藏公告卡片
```

:::

## 📝 内容配置

### 公告内容文件

编辑 `data/announcement.yaml` 文件设置公告内容：

::: code-group

```yaml [基础公告📢]
title: "📢 站点公告"
content: |
  欢迎来到我的博客！
  这里记录着我的技术成长历程。
icon: "bell"
```

```yaml [配置说明📝]
title: "公告标题"     # ◀️ 卡片标题，支持 emoji
content: |           # ◀️ 公告内容，支持 Markdown
  多行内容使用 | 符号
  支持 **加粗**、*斜体* 等格式
icon: "图标名称"      # ◀️ Bootstrap Icons 图标名
```

:::

### 多公告轮播配置

支持多条公告自动轮播显示：

::: code-group

```yaml [多条公告🔄]
announcements:
  - title: "🎉 新功能上线"
    content: |
      评论系统已接入，欢迎交流讨论！
    icon: "chat-dots"
    date: "2026-03-01"
    
  - title: "📢 维护通知"
    content: |
      本站将于本周日凌晨进行例行维护。
    icon: "tools"
    date: "2026-02-28"
    
  - title: "📝 更新日志"
    content: |
      新增暗色模式、优化移动端体验。
    icon: "journal-text"
    date: "2026-02-25"
```

:::

## 🎨 高级内容格式

### Markdown 支持

公告内容支持完整的 Markdown 语法：

::: code-group

```yaml [Markdown 格式✨]
title: "📢 重要通知"
content: |
  ## 🎉 站点升级完成
  
  本次更新内容包括：
  - **新主题**：更换为 Hugo-Teek 主题
  - **新功能**：新增搜索、评论功能
  - **优化**：提升页面加载速度
  
  [查看详情](/about/website/)
icon: "stars"
```

```yaml [链接和强调🔗]
title: "🔗 友情链接"
content: |
  欢迎访问我的其他站点：
  
  - [GitHub](https://github.com/xxx)
  - [技术文档](/docs/)
  - **邮箱联系**：xxx@example.com
icon: "link-45deg"
```

:::

### 图标配置

使用 Bootstrap Icons 美化公告标题：

::: code-group

```yaml [常用图标🎨]

icon: "clock"          # ◀️ 时钟 - 时间相关
icon: "megaphone"      # ◀️ 喇叭 - 重要公告
```

:::

## 🔧 常见问题

### 公告不显示

::: code-group

```markdown [问题：公告不显示❌]
原因1：开关未启用
解决：检查 homepage.toml 中 enabled = true

原因2：YAML 语法错误
解决：
1️⃣ 使用 YAML 验证器检查语法
2️⃣ 注意缩进使用空格而非 Tab
3️⃣ 特殊字符需要转义

原因3：文件路径错误
解决：确认文件位于 hugo-teek-site/data/announcement.yaml
```

:::

### 格式显示异常

::: code-group

```markdown [问题：Markdown 不渲染❌]
原因：YAML 格式问题
解决：
1️⃣ 使用 | 符号表示多行文本
2️⃣ 确保正确缩进
3️⃣ 避免混用空格和 Tab

示例：
content: |
  ## 标题
  - 列表项1
  - 列表项2
```

:::
