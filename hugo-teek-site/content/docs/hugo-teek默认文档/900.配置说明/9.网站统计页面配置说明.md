## 📊 网站统计页面概述

Hugo-Teek 主题提供完善的网站统计功能，支持多种统计服务接入，包括不蒜子访问统计、Umami网站分析、51.LA统计组件等，帮助博主全面了解站点访问情况。

### ✨ 统计功能特性

::: tip 🎨 展示特性
- 📈 实时数据：显示今日/昨日/本月访客数和访问量
- 🎯 多维度：支持页面浏览量(PV)和独立访客数(UV)统计
- 🌐 第三方集成：支持Umami、不蒜子、51.LA等多种统计服务
- 📱 卡片展示：侧边栏站点统计卡片自动展示站点数据
:::

::: info ⚙️ 配置特性
- 🔌 灵活开关：每种统计服务均可独立启用/禁用
- 🎨 样式定制：支持数字格式、主题颜色等自定义
- 🔗 外部链接：可链接到详细的统计后台查看更多信息
- 📊 数据合并：支持多种统计服务同时使用
:::

## 📈 不蒜子统计配置

### 启用不蒜子统计

不蒜子是一款轻量级的网站访问统计工具，支持显示今日/昨日/本月的访客数和访问量：

::: code-group

```toml [不蒜子统计配置📊]
# config/themes/hugo-teek/services.toml
[analytics]
  [analytics.busuanzi]
    enabled = true                    # ◀️ 总开关：启用/禁用不蒜子统计
    url = "https://busuanzi-eo.xxdevops.cn/busuanzi.eo.js"
    api = "https://busuanzi-eo.xxdevops.cn/api/count"
    type = "eo"                       # ◀️ 版本类型：eo（推荐）| classic
    prefix = "busuanzi"
    style = "comma"                   # ◀️ 数字格式：comma（千分位）| none
    pjax = false
    page_pv = true                    # ◀️ 显示页面浏览量
    site_pv = true                    # ◀️ 显示站点总浏览量
    site_uv = true                    # ◀️ 显示站点独立访客数
```

```toml [配置字段说明📝]
[analytics.busuanzi]
  enabled = true          # ◀️ 总开关，设为 false 禁用不蒜子统计
  type = "eo"             # ◀️ EO版本支持更多功能，classic为传统版本
  style = "comma"         # ◀️ comma=千分位格式(1,234)，none=纯数字(1234)
  page_pv = true          # ◀️ 是否显示当前页面浏览量
  site_pv = true          # ◀️ 是否显示站点总浏览量
  site_uv = true          # ◀️ 是否显示站点独立访客数
```

:::

### 不蒜子显示效果

::: code-group

```yaml [页脚显示效果📋]
今日访客数 1 | 今日访问量 387 | 昨日访客数 1 | 昨日访问量 1724 | 本月访客数 1 | 本月访问量 2110
```

```yaml [页面显示效果📋]
本文总阅读量 128 次
```

:::

### 不蒜子字段详细说明

| 配置项 | 必填 | 说明 | 可选值 |
|--------|------|------|--------|
| `enabled` | ✅ | 总开关 | `true` / `false` |
| `type` | ❌ | 版本类型 | `eo`（推荐）/ `classic` |
| `style` | ❌ | 数字显示格式 | `comma`（千分位）/ `none` |
| `url` | ✅ | JS文件地址 | 不蒜子服务URL |
| `api` | ✅ | API端点地址 | 计数API地址 |
| `page_pv` | ❌ | 页面浏览量 | `true` / `false` |
| `site_pv` | ❌ | 站点总浏览量 | `true` / `false` |
| `site_uv` | ❌ | 站点独立访客 | `true` / `false` |

## 🌐 Umami网站分析配置

### 启用Umami分析

Umami是一款开源的网站分析工具，提供详细的访问数据分析：

::: code-group

```toml [Umami配置🌐]
# config/themes/hugo-teek/services.toml
[analytics]
  [analytics.umami]
    enabled = true        # ◀️ 总开关：启用/禁用 Umami 分析
    url = "https://umami.xxdevops.cn"
    websiteId = "a50c01fa-f866-488d-a328-3ac73aa639f7"
```

```toml [配置说明📝]
[analytics.umami]
  enabled = true          # ◀️ 设为 true 启用 Umami 统计代码注入
  url = "https://..."     # ◀️ Umami 服务地址
  websiteId = "..."       # ◀️ 网站ID，在 Umami 后台获取
```

:::

### Umami后台入口配置

在导航栏添加Umami统计后台入口，方便快速查看详细数据：

::: code-group

```toml [导航栏配置🔗]
# config/teek-plugins/menus.toml
[[main]]
  name = "📊 统计后台"
  parent = "about"
  url = "https://umami.xxdevops.cn/"
  weight = 33.0
  [main.params]
    icon = "bar-chart"
    external = true       # ◀️ 标记为外部链接
    target = "_blank"     # ◀️ 在新窗口打开
```

```toml [独立菜单项🔗]
[[main]]
  name = "📊 网站统计"
  url = "https://umami.xxdevops.cn/"
  weight = 10
  [main.params]
    icon = "bar-chart"
    external = true
    target = "_blank"
```

:::

## 📋 51.LA统计组件配置

### 启用51.LA统计组件

51.LA提供可视化统计组件，可在页脚展示精美的访问统计：

::: code-group

```toml [51.LA组件配置📋]
# config/themes/hugo-teek/services.toml
[footer]
  [footer.analytics]
    la_widget_enabled = true    # ◀️ 总开关：启用/禁用 LA 统计组件
    la_widget_id = "3LmZHLhDZIDpMaT0"
    la_widget_theme = "#1690FF,#333333,#999999,#007BFF,#FFFFFF,#1690FF,12"
```

```toml [配置说明📝]
[footer.analytics]
  la_widget_enabled = true                    # ◀️ 设为 true 显示统计组件
  la_widget_id = "你的组件ID"                  # ◀️ 在 51.LA 后台获取
  la_widget_theme = "#1690FF,#333333,..."     # ◀️ 主题颜色配置（7个颜色值）
```

:::

### 51.LA主题颜色说明

::: code-group

```yaml [主题颜色格式🎨]
la_widget_theme 格式：颜色1,颜色2,颜色3,颜色4,颜色5,颜色6,字体大小

颜色1: 主色调（标题、边框）
颜色2: 数字颜色
颜色3: 标签颜色
颜色4: 链接颜色
颜色5: 背景颜色
颜色6: 悬停颜色
字体大小: 数字（单位：像素）
```

```toml [常用主题配置🎨]
# 蓝色主题（默认）
la_widget_theme = "#1690FF,#333333,#999999,#007BFF,#FFFFFF,#1690FF,12"

# 深色主题
la_widget_theme = "#333333,#FFFFFF,#CCCCCC,#1690FF,#1A1A1A,#333333,12"

# 绿色主题
la_widget_theme = "#52C41A,#333333,#999999,#389E0D,#FFFFFF,#52C41A,12"
```

:::

## 📊 站点统计卡片配置

### 启用站点统计卡片

站点统计卡片展示在首页侧边栏，显示文章数、分类数、标签数等站点数据：

::: code-group

```toml [站点统计卡片开关📊]
# config/themes/hugo-teek/homepage.toml
[params.homeSidebar.siteInfo]
    enabled = true        # ◀️ 设为 true 显示站点统计卡片
```

```toml [完整配置示例📋]
# ========================================
# 首页配置 (Homepage Configuration)
# ========================================
[params.homeSidebar]

  # ========================================
  # 站点统计卡片 (Site Info Card)
  # ========================================
  # 显示站点统计信息（文章数、分类数等）
  [params.homeSidebar.siteInfo]
    enabled = true        # ◀️ 开关：启用/禁用站点统计卡片
```

:::

## 🔧 常见问题

### 统计数据不显示或显示异常

::: code-group

```markdown [问题1：不蒜子统计不显示❌]
原因：enabled 未启用或网络问题
解决：
1️⃣ 检查 services.toml 中 [analytics.busuanzi] enabled = true
2️⃣ 确认 url 和 api 地址可正常访问
3️⃣ 检查浏览器控制台是否有 JS 加载错误
4️⃣ 确认网络连接正常，不蒜子服务可达
```

```markdown [问题2：Umami统计不生效❌]
原因：websiteId 错误或配置未生效
解决：
1️⃣ 检查 websiteId 是否与 Umami 后台一致
2️⃣ 确认 url 地址正确（包含 https://）
3️⃣ 在 Umami 后台确认网站已添加
4️⃣ 检查浏览器开发者工具 Network 面板是否有请求
```

```markdown [问题3：51.LA组件不显示❌]
原因：widget_id 错误或 enabled 未开启
解决：
1️⃣ 检查 la_widget_enabled = true
2️⃣ 确认 la_widget_id 从 51.LA 后台正确获取
3️⃣ 检查主题颜色格式是否正确（7个颜色值）
4️⃣ 确认 51.LA 统计代码已正确部署
```

```markdown [问题4：站点统计卡片不显示❌]
原因：enabled 设为 false 或启动日期未配置
解决：
1️⃣ 检查 homepage.toml 中 siteInfo.enabled = true
2️⃣ 确认 site.toml 中 siteStartDate 已配置
3️⃣ 检查首页布局是否包含侧边栏
4️⃣ 重启开发服务器：make dev
```

:::

### 配置生效检查清单

```markdown
✅ 配置文件路径正确
✅ TOML 语法无误（使用在线 TOML 验证器检查）
✅ 启用开关设为 true
✅ 必填字段（url、api、websiteId等）已填写
✅ 站点启动日期格式正确（YYYY-MM-DD）
✅ 运行 make dev 或 make build 重新合并配置
✅ 浏览器强制刷新（Ctrl+F5）清除缓存
```
