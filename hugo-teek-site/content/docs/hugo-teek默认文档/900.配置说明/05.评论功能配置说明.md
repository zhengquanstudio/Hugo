## 💬 评论功能配置概述

Hugo-Teek 主题内置强大的评论系统支持，目前默认集成 [Twikoo](https://twikoo.js.org/) 评论系统。Twikoo 是一个简洁、安全、免费的静态网站评论系统，支持多种部署方式。

### ✨ 评论系统特性

::: tip 🎨 功能特性
- 🚀 多种部署方式：支持 Vercel、CloudBase、Docker 等
- 💬 实时通知：支持邮件、微信、QQ 等通知方式
- 🎨 丰富的交互：支持表情包、图片上传、Markdown 语法
- 🔒 安全可靠：支持垃圾评论过滤、关键词屏蔽
- 📱 响应式设计：完美适配移动端设备
:::

::: info ⚙️ 主题集成特性
- 一键开关：通过 `enabled` 参数快速启用/禁用
- 自动初始化：页面加载时自动初始化评论系统
- 最近评论：首页侧边栏支持显示最新评论列表
- 管理员标识：支持隐藏管理员设置按钮
:::

## 📁 配置文件路径

Hugo-Teek 采用分层配置结构，评论系统配置文件位于以下位置：

::: code-group

```toml [通用服务配置📄]
# hugo-teek-site/config/teek-plugins/services.toml
# 所有主题共用的服务配置
# 推荐在此文件中进行评论配置
```

```toml [主题特定配置📄]
# hugo-teek-site/config/themes/hugo-teek/services.toml
# 仅当前主题使用的服务配置
# 用于覆盖通用配置
```

:::

### 配置优先级

```
配置合并顺序（优先级从低到高）：
1️⃣ config/themes/{theme}/services.toml（主题默认配置）
2️⃣ config/teek-plugins/services.toml（通用配置）
3️⃣ 运行时通过管理后台修改的配置
```

## 📝 基础评论配置

### 启用评论系统

最简单的评论配置，只需设置启用开关和 Twikoo 环境 ID：

::: code-group

```toml [基础配置📝]
[comment]
  enabled = true                              # ◀️ 总开关：启用评论系统
  provider = "twikoo"                         # ◀️ 评论提供商，目前支持 twikoo
  envId = "https://twikoo.example.com/"       # ◀️ Twikoo 服务器地址，记得替换自己部署的
```

```toml [配置说明📝]
[comment]
  enabled = true        # ◀️ 启用/禁用评论系统
  provider = "twikoo"   # ◀️ 评论系统提供商
  envId = "..."         # ◀️ Twikoo 环境 ID 或自定义域名
```

:::

### 评论配置字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `enabled` | ✅ | 评论系统总开关，`true` 启用，`false` 禁用 |
| `provider` | ✅ | 评论系统提供商，目前仅支持 `twikoo` |
| `envId` | ✅ | Twikoo 服务器地址（环境 ID 或自定义域名） |
| `version` | ❌ | Twikoo 版本号，如 `1.6.41` |
| `hideAdmin` | ❌ | 是否隐藏管理员设置按钮，`true` 隐藏 |

## 🚀 Twikoo 部署指南

### 方式一：Vercel 部署（推荐）

Vercel 部署是最简单快捷的方式，免费且稳定：

::: code-group

```bash [部署步骤🚀]
# 1️⃣  fork Twikoo 仓库
# 访问 https://github.com/imaegoo/twikoo 点击 Fork

# 2️⃣ 在 Vercel 导入项目
# 访问 https://vercel.com/new
# 选择你 fork 的 twikoo 仓库

# 3️⃣ 配置环境变量
# 添加 MONGODB_URI 变量
# 格式：mongodb+srv://用户名:密码@集群地址/数据库名

# 4️⃣ 获取环境 ID
# 部署完成后，Vercel 会分配域名
# 如：https://twikoo-xxx.vercel.app
```

```toml [配置示例📝]
[comment]
  enabled = true
  provider = "twikoo"
  envId = "https://twikoo-xxx.vercel.app"  # ◀️ Vercel 分配的域名
```

:::

### 方式二：腾讯云 CloudBase 部署

适合国内用户，访问速度更快：

::: code-group

```yaml [部署步骤☁️]
1️⃣ 登录腾讯云 CloudBase 控制台
   访问：https://console.cloud.tencent.com/tcb

2️⃣ 创建新环境
   选择【新建环境】-> 选择免费版

3️⃣ 安装 Twikoo 云函数
   进入环境 -> 云函数 -> 新建云函数
   选择模板：Twikoo 评论系统

4️⃣ 获取环境 ID
   在环境设置中查看环境 ID
   格式：twikoo-xxx
```

```toml [配置示例📝]
[comment]
  enabled = true
  provider = "twikoo"
  envId = "twikoo-xxx"  # ◀️ CloudBase 环境 ID
```

:::

### 方式三：Docker 部署

适合有服务器的用户，自主可控：

::: code-group

```yaml [docker-compose.yml🐳]
version: '3'
services:
  twikoo:
    image: imaegoo/twikoo:latest
    container_name: twikoo
    restart: always
    ports:
      - "8080:8080"           # ◀️ 映射端口
    environment:
      - MONGODB_URI=mongodb://mongo:27017/twikoo
      - TWIKOO_THROTTLE=1000   # ◀️ 限流配置（毫秒）
    volumes:
      - ./data:/app/data       # ◀️ 数据持久化
```

```toml [配置示例📝]
[comment]
  enabled = true
  provider = "twikoo"
  envId = "https://twikoo.example.com"  # ◀️ 你的服务器域名
```

:::

### 隐藏管理员标识

如果你不想在评论区显示管理员设置按钮（齿轮图标），可以开启此选项：

::: code-group

```toml [隐藏管理按钮🔒]
[comment]
  enabled = true
  provider = "twikoo"
  envId = "https://twikoo.example.com/"
  hideAdmin = true        # ◀️ 隐藏管理员设置按钮
```

```css [生效效果🎨]
/* 隐藏 Twikoo 管理设置按钮（齿轮图标） */
.tk-admin {
  display: none !important;
}
```

## 🏠 首页最近评论卡片

### 启用最近评论卡片

首页侧边栏可以显示最新评论列表，增加站点互动性：

::: code-group

```toml [启用最近评论📋]
[params.homeSidebar.recentComments]
  enabled = true        # ◀️ 启用最近评论卡片
  pageSize = 5          # ◀️ 每页显示评论数量
  totalSize = 50        # ◀️ 总共获取评论数量
```

```toml [配置路径📄]
# 配置文件位置：
# config/themes/hugo-teek/homepage.toml
# 或
# config/teek-plugins/homepage.toml
```

:::

### 最近评论配置字段说明

| 字段 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `enabled` | ✅ | `true` | 最近评论卡片开关 |
| `pageSize` | ❌ | `5` | 每页显示的评论数量 |
| `totalSize` | ❌ | `50` | 从服务器获取的评论总数 |

### 依赖关系

::: warning ⚠️ 注意事项
最近评论卡片依赖评论系统，请确保：
1️⃣ `services.toml` 中 `comment.enabled = true`
2️⃣ Twikoo 服务正常运行
3️⃣ 网络可以访问 Twikoo 服务器
:::

## 🔧 常见问题

### 评论不显示或加载失败

::: code-group

```markdown [问题1：评论区不显示❌]
原因：comment.enabled 设置为 false 或 envId 配置错误
解决：
1️⃣ 检查 services.toml 中 enabled = true
2️⃣ 确认 envId 地址正确且可访问
3️⃣ 检查浏览器控制台是否有报错
```

```markdown [问题2：评论加载失败❌]
原因：Twikoo 服务未启动或跨域问题
解决：
1️⃣ 访问 envId 地址确认服务正常
2️⃣ 检查 Twikoo 云函数/容器是否运行
3️⃣ 查看 Twikoo 后台是否有错误日志
```

```markdown [问题3：最近评论不显示❌]
原因：homepage.toml 配置错误或依赖未满足
解决：
1️⃣ 检查 homepage.toml 中 enabled = true
2️⃣ 确认 comment.enabled 也为 true
3️⃣ 检查网络是否可以访问 Twikoo 服务器
```

:::

### Twikoo 配置检查清单

```markdown
✅ MongoDB 数据库连接正常
✅ Twikoo 云函数/服务已部署并运行
✅ envId 配置正确（区分环境 ID 和自定义域名）
✅ 域名已添加到 Twikoo 安全域名白名单
✅ Hugo 配置文件中 enabled = true
✅ 重启开发服务器后生效
```
