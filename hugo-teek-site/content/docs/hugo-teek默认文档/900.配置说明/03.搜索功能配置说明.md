## 🔍 搜索功能概述

Hugo-Teek 主题内置强大的离线全文搜索功能，基于 [FlexSearch](https://github.com/nextapps-de/flexsearch) 搜索引擎实现。无需依赖第三方服务，即可在本地提供极速的搜索体验。

### ✨ 搜索特性

::: tip ⚡ 性能特性
- 离线搜索：无需网络请求，本地即时响应
- 极速检索：基于 FlexSearch 的高性能索引
- 全文匹配：支持标题、内容、标签多维度搜索
- 智能高亮：搜索结果关键词自动高亮显示
:::

::: info 🎨 交互特性
- 快捷键唤起：支持 `Ctrl+K` / `Cmd+K` 快速打开搜索
- 搜索历史：自动保存最近搜索记录
- 键盘导航：支持方向键选择、回车确认、ESC 关闭
- 移动端适配：完美支持手机端搜索体验
:::

## 📝 基础搜索配置

### 启用/禁用搜索功能

最简单的搜索配置，只需设置 enabled 字段：

::: code-group

```toml [启用搜索✅]
# hugo-teek-site/config/themes/hugo-teek/params.toml
[search]
    enabled = true
    provider = "flexsearch"
```

```toml [禁用搜索❌]
# hugo-teek-site/config/themes/hugo-teek/params.toml
[search]
    enabled = false
```

```toml [hugo-simple 主题配置📝]
# hugo-simple 主题默认关闭搜索
[params.search]
    enabled = false
```

:::

### 搜索配置字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `enabled` | ✅ | 是否启用搜索功能，`true` 开启，`false` 关闭 |
| `provider` | ❌ | 搜索引擎类型，默认为 `flexsearch` |

## 🚀 生成搜索索引

### 自动生成索引

开发模式下会自动生成搜索索引：

::: code-group

```bash [开发模式自动生成🚀]
make dev
# 会自动执行 gen-search-index 生成索引
```

```bash [手动生成索引🔍]
make gen-search-index
```

```bash [快速启动（无搜索）⚡]
make quick
# 跳过搜索索引生成，快速预览
```

:::

### 索引文件位置

搜索索引生成后会保存在以下位置：

```
hugo-teek-site/
└── static/
    └── data/
        └── search/
            └── index.json    # ◀️ FlexSearch 索引文件
```

::: warning ⚠️ 注意事项
- 索引文件会在构建时自动生成，无需手动创建
- 如果搜索不工作，尝试手动运行 `make gen-search-index`
- 清理构建时索引会被删除，需要重新生成
:::

## 🛠️ 常用命令

### 搜索相关 Make 命令

::: code-group

```bash [生成搜索索引🔍]
make gen-search-index
# 生成 FlexSearch 搜索索引
```

```bash [快速重建（复用索引）⚡]
make rebuild-fast
# 清理但保留搜索索引，快速重建
```

```bash [清理但保留索引🧹]
make clean-keep-index
# 清理构建文件但保留搜索索引
```

```bash [极速构建（无索引）🚀]
make build-fastest
# 跳过所有非必要步骤，不包含搜索索引
```

:::

### 命令对比

| 命令 | 搜索索引 | 适用场景 | 构建时间 |
|------|----------|----------|----------|
| `make dev` | ✅ 生成 | 完整开发环境 | ~60秒 |
| `make quick` | ❌ 跳过 | 快速预览 | ~10秒 |
| `make rebuild-fast` | ✅ 复用 | 增量开发 | ~12秒 |
| `make build-fastest` | ❌ 无 | 极速预览 | ~10秒 |

## 🔧 常见问题

### 搜索不工作或无法显示

::: code-group

```markdown [问题1：搜索按钮不显示❌]
原因：搜索功能未启用或配置错误
解决：
1️⃣ 检查 config/themes/hugo-teek/params.toml
2️⃣ 确认 [search] enabled = true
3️⃣ 重启开发服务器：make dev
```

```markdown [问题2：搜索无结果❌]
原因：搜索索引未生成或为空
解决：
1️⃣ 手动生成索引：make gen-search-index
2️⃣ 检查索引文件是否存在：
   hugo-teek-site/static/data/search/index.json
3️⃣ 重新构建：make dev
```

```markdown [问题3：搜索结果不准确❌]
原因：索引数据过期或内容未更新
解决：
1️⃣ 清理并重新生成索引
2️⃣ 执行：make clean && make dev
3️⃣ 或使用：make rebuild-fast
```

:::

## 💡 最佳实践

### 搜索优化建议

::: tip 🎯 性能优化
1. **定期重建索引**：内容更新较多时，建议定期执行 `make gen-search-index`
2. **使用快速重建**：开发时使用 `make rebuild-fast` 复用索引，节省时间
3. **清理缓存**：搜索异常时尝试清理浏览器缓存或执行 `make clean`
:::

::: tip 📝 内容优化
1. **优化文章标题**：清晰的标题有助于提高搜索准确性
2. **完善标签**：为文章添加准确的标签，提升标签搜索效果
3. **控制文章长度**：适当分段，有利于内容匹配
:::
