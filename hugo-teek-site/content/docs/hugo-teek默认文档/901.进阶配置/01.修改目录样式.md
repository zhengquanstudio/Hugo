## 📑 目录样式修改概述

Hugo-Teek 主题支持通过 CSS 变量自定义文档落地页左侧目录的样式，包括文本颜色、背景色、悬停效果等。通过统一的 CSS 变量系统，可以轻松实现亮色/暗色模式下的样式适配。

### ✨ 可自定义的样式项

::: tip 🎨 样式定制范围
- 文本颜色：普通文本、选中文本、悬停文本
- 背景颜色：容器背景、选中背景、悬停背景
- 子目录样式：下拉后的文本色、悬停样式、选中样式
- 边框与阴影：分隔线、圆角、阴影效果
:::

::: info ⚙️ 技术特性
- CSS 变量管理：使用 CSS 自定义属性实现样式统一管理
- 主题适配：支持亮色/暗色模式自动切换
- 层级样式：父目录与子目录样式独立配置
- 响应式：移动端样式自动适配
:::

## 📁 样式文件路径

目录样式配置文件位于以下位置：

::: code-group

```scss [通用样式配置📄]
/* hugo-teek-site/assets/css/custom/sidebar.scss */
/* 所有主题共用的目录样式配置 */
/* 推荐在此文件中进行样式配置 */
```

```scss [主题特定样式📄]
/* hugo-teek-site/assets/themes/hugo-teek/css/sidebar.scss */
/* 仅当前主题使用的目录样式配置 */
/* 用于覆盖通用配置 */
```

```scss [变量定义文件📄]
/* hugo-teek-site/assets/css/variables.scss */
/* 全局 CSS 变量定义文件 */
/* 定义目录样式的基础变量 */
```

:::

### 配置优先级

```
样式合并顺序（优先级从低到高）：
1️⃣ assets/css/variables.scss（基础变量定义）
2️⃣ assets/themes/{theme}/css/sidebar.scss（主题默认样式）
3️⃣ assets/css/custom/sidebar.scss（用户自定义样式）
```

## 🎨 CSS 变量说明

### 基础变量定义

首先需要在变量文件中定义目录样式的基础 CSS 变量：

::: code-group

```scss [变量定义📝]
/* hugo-teek-site/assets/css/variables.scss */

/* ================================
   📑 目录样式变量
   ================================ */

/* ☀️ 亮色模式变量 */
:root {
  /* 容器背景 */
  --sidebar-bg: #ffffff;                    /* ◀️ 侧边栏容器背景色 */
  --sidebar-border-color: #e5e7eb;          /* ◀️ 侧边栏边框颜色 */
  
  /* 文本颜色 */
  --sidebar-text-color: #374151;            /* ◀️ 普通文本颜色 */
  --sidebar-text-hover: #111827;            /* ◀️ 悬停文本颜色 */
  --sidebar-text-active: #2563eb;           /* ◀️ 选中文本颜色 */
  
  /* 背景颜色 */
  --sidebar-item-hover-bg: #f3f4f6;         /* ◀️ 悬停背景色 */
  --sidebar-item-active-bg: #eff6ff;        /* ◀️ 选中背景色 */
  
  /* 子目录样式 */
  --sidebar-child-text-color: #4b5563;      /* ◀️ 子目录文本颜色 */
  --sidebar-child-text-hover: #1f2937;      /* ◀️ 子目录悬停文本色 */
  --sidebar-child-text-active: #2563eb;     /* ◀️ 子目录选中文本色 */
  --sidebar-child-bg: #f9fafb;              /* ◀️ 子目录背景色 */
}

/* 🌆 暗色模式变量 */
[data-theme="dark"] {
  /* 容器背景 */
  --sidebar-bg: #1f2937;                    /* ◀️ 侧边栏容器背景色 */
  --sidebar-border-color: #374151;          /* ◀️ 侧边栏边框颜色 */
  
  /* 文本颜色 */
  --sidebar-text-color: #d1d5db;            /* ◀️ 普通文本颜色 */
  --sidebar-text-hover: #f9fafb;            /* ◀️ 悬停文本颜色 */
  --sidebar-text-active: #60a5fa;           /* ◀️ 选中文本颜色 */
  
  /* 背景颜色 */
  --sidebar-item-hover-bg: #374151;         /* ◀️ 悬停背景色 */
  --sidebar-item-active-bg: #1e3a8a;        /* ◀️ 选中背景色 */
  
  /* 子目录样式 */
  --sidebar-child-text-color: #9ca3af;      /* ◀️ 子目录文本颜色 */
  --sidebar-child-text-hover: #e5e7eb;      /* ◀️ 子目录悬停文本色 */
  --sidebar-child-text-active: #60a5fa;     /* ◀️ 子目录选中文本色 */
  --sidebar-child-bg: #111827;              /* ◀️ 子目录背景色 */
}
```

```scss [变量说明📝]
/* 变量命名规范：
 * --sidebar-{元素}-{状态}-{属性}
 * 
 * 元素：bg(背景)、text(文本)、item(菜单项)、child(子目录)
 * 状态：hover(悬停)、active(选中)、无(默认状态)
 * 属性：color(颜色)、bg(背景色)
 */
```

:::

### 变量使用示例

::: code-group

```scss [基础使用📝]
/* 在样式文件中使用变量 */
.simple-sidebar {
  background-color: var(--sidebar-bg);
  border-right: 1px solid var(--sidebar-border-color);
}

.simple-sidebar__link {
  color: var(--sidebar-text-color);
}

.simple-sidebar__link:hover {
  color: var(--sidebar-text-hover);
  background-color: var(--sidebar-item-hover-bg);
}

.simple-sidebar__item.is-active .simple-sidebar__link {
  color: var(--sidebar-text-active);
  background-color: var(--sidebar-item-active-bg);
}
```

```scss [进阶使用📝]
/* 使用 fallback 值确保兼容性 */
.simple-sidebar {
  background-color: var(--sidebar-bg, #ffffff);
}

/* 嵌套使用变量 */
.simple-sidebar__children {
  background-color: var(--sidebar-child-bg, var(--sidebar-bg));
}
```

## 📝 目录样式配置

### 容器样式配置

配置侧边栏容器的背景色、边框等基础样式：

::: code-group

```scss [容器基础样式📝]
/* hugo-teek-site/assets/css/custom/sidebar.scss */

/* ================================
   📦 侧边栏容器样式
   ================================ */

.simple-sidebar {
  /* 基础布局 */
  display: flex;
  flex-direction: column;
  width: 280px;                             /* ◀️ 侧边栏宽度 */
  height: 100vh;                            /* ◀️ 占满视口高度 */
  
  /* 背景与边框 */
  background-color: var(--sidebar-bg);      /* ◀️ 使用变量定义背景色 */
  border-right: 1px solid var(--sidebar-border-color);
  
  /* 滚动行为 */
  overflow-y: auto;
  scrollbar-width: thin;
  scrollbar-color: var(--sidebar-border-color) transparent;
}

/* 滚动条样式 */
.simple-sidebar::-webkit-scrollbar {
  width: 6px;
}

.simple-sidebar::-webkit-scrollbar-thumb {
  background-color: var(--sidebar-border-color);
  border-radius: 3px;
}
```

```scss [容器进阶样式📝]
/* 添加阴影效果 */
.simple-sidebar {
  box-shadow: 2px 0 8px rgba(0, 0, 0, 0.05);
}

/* 🌆 暗色模式阴影 */
[data-theme="dark"] .simple-sidebar {
  box-shadow: 2px 0 8px rgba(0, 0, 0, 0.2);
}

/* 圆角配置 */
.simple-sidebar {
  border-radius: 0 12px 12px 0;             /* ◀️ 右侧圆角 */
}
```

### 菜单项文本样式

配置目录菜单项的文本颜色、悬停效果和选中状态：

::: code-group

```scss [文本颜色配置📝]
/* ================================
   📝 菜单项文本样式
   ================================ */

.simple-sidebar__link {
  /* 基础文本样式 */
  color: var(--sidebar-text-color);         /* ◀️ 普通文本颜色 */
  font-size: 14px;                          /* ◀️ 字体大小 */
  font-weight: 400;                         /* ◀️ 字体粗细 */
  line-height: 1.5;
  text-decoration: none;
  
  /* 过渡动画 */
  transition: color 0.2s ease, background-color 0.2s ease;
}

/* 悬停状态 */
.simple-sidebar__link:hover {
  color: var(--sidebar-text-hover);         /* ◀️ 悬停文本颜色 */
}

/* 选中状态 */
.simple-sidebar__item.is-active .simple-sidebar__link {
  color: var(--sidebar-text-active);        /* ◀️ 选中文本颜色 */
  font-weight: 500;                         /* ◀️ 选中时加粗 */
}
```

### 背景色配置

配置菜单项的悬停背景色和选中背景色：

::: code-group

```scss [背景色配置📝]
/* ================================
   🎨 菜单项背景样式
   ================================ */

.simple-sidebar__item-inner {
  /* 基础样式 */
  padding: 8px 16px;
  border-radius: 6px;                       /* ◀️ 圆角 */
  margin: 2px 8px;
  
  /* 过渡动画 */
  transition: background-color 0.2s ease;
}

/* 悬停背景 */
.simple-sidebar__item:hover .simple-sidebar__item-inner,
.simple-sidebar__item-inner:hover {
  background-color: var(--sidebar-item-hover-bg);

/* 选中背景 */
.simple-sidebar__item.is-active .simple-sidebar__item-inner {
  background-color: var(--sidebar-item-active-bg);

/* 选中+悬停组合状态 */
.simple-sidebar__item.is-active:hover .simple-sidebar__item-inner {
  background-color: var(--sidebar-item-active-bg);
  opacity: 0.9;
}
```

### 子目录容器样式

配置子目录下拉后的容器样式：

::: code-group

```scss [子目录容器📝]
/* ================================
   🌳 子目录容器样式

  border-left: 2px solid var(--sidebar-border-color);

.simple-sidebar__item.has-children.is-expanded .simple-sidebar__children {
  display: block;
  opacity: 1;
}

/* 收起状态 */
.simple-sidebar__item.has-children:not(.is-expanded) .simple-sidebar__children {
  display: none;
  opacity: 0;
}
```

## 🔧 常见问题

### 样式不生效

::: code-group

```markdown [问题1：样式不生效❌]
原因：CSS 文件未正确引入或变量未定义
解决：
1️⃣ 检查 variables.scss 是否正确导入
2️⃣ 确认 CSS 变量命名是否正确
3️⃣ 清除浏览器缓存后刷新
4️⃣ 检查浏览器开发者工具查看实际应用的样式
```
