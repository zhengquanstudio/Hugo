## 🎉 Hugo-Teek 是什么？

**Hugo-Teek** 是一款基于 VitePress Teek 主题风格移植的 Hugo 静态网站主题，专为技术博客、文档站点和个人知识库设计。它将 VitePress 的精美设计与 Hugo 的极速构建能力完美结合，为你提供【开箱即用】的博客解决方案。

### 🌟 项目定位

::: tip 💡 我们的目标
打造一个【零成本、高颜值、易维护】的个人博客系统，让每个人都能轻松拥有属于自己的技术博客。
:::

## 🚀 零成本部署方案：CNB + EO

### 什么是 CNB + EO？

::: tip 🔗 技术栈说明
- **CNB**（Codeup Native Build）：CSDN 推出的云原生构建平台，提供免费 CI/CD 能力
- **EO**（EdgeOne）：腾讯云边缘安全加速平台，提供免费静态网站托管
:::

## 📋 零成本部署详细步骤

### 第一步：准备代码仓库

::: code-group

```bash [1️⃣ Fork 项目📦]
# 访问项目仓库
https://cnb.cool/yuwen-gueen/hugo-teeker-theme

# 点击 Fork 按钮，将项目复制到你的 CNB 账户
```

```bash [2️⃣ 克隆到本地💻]
git clone https://cnb.cool/你的用户名/hugo-teeker-theme.git
cd hugo-teeker-theme
```

:::

### 第二步：配置 CNB CI/CD

::: code-group

```yaml [配置文件路径📄]
# 文件: .cnb.yml（已内置）
# 位于项目根目录
```

```yaml [CI 配置示例⚙️]
# 🥳 CNB 自动构建配置
# 🔗 自动构建 Hugo 站点并部署到 EO

stages:
  - build
  - deploy

variables:
  HUGO_VERSION: "0.150.0"

build:
  stage: build
  image: golang:1.21
  script:
    # ◀️ 安装 Hugo Extended
    - wget https://github.com/gohugoio/hugo/releases/download/v${HUGO_VERSION}/hugo_extended_${HUGO_VERSION}_Linux-64bit.tar.gz
    - tar -xzf hugo_extended_${HUGO_VERSION}_Linux-64bit.tar.gz
    - mv hugo /usr/local/bin/
    # ◀️ 构建站点
    - cd hugo-teek-site
    - hugo --minify
  artifacts:
    paths:
      - hugo-teek-site/public/
    expire_in: 1 hour

deploy:
  stage: deploy
  image: python:3.9
  script:
    # ◀️ 安装 EO CLI 工具
    - pip install tccli
    # ◀️ 配置 EO 凭证（在 CNB 设置中添加环境变量）
    - tccli configure set secretId $EO_SECRET_ID
    - tccli configure set secretKey $EO_SECRET_KEY
    - tccli configure set region ap-guangzhou
    # ◀️ 部署到 EO 静态存储
    - tccli edgeone upload --ZoneId $EO_ZONE_ID --Path "/" --LocalPath ./hugo-teek-site/public/
  only:
    - main
```

:::

### 第三步：配置 EO 静态托管

::: code-group

```markdown [EO 配置步骤☁️]
1️⃣ 注册腾讯云账号（已有可跳过）
   访问: https://cloud.tencent.com/

2️⃣ 开通 EdgeOne 服务
   搜索「EdgeOne」→ 立即开通 → 选择「免费版」

3️⃣ 添加站点
   点击「添加站点」→ 输入你的域名
   或选择「使用 EO 提供的域名」（推荐新手）

4️⃣ 开启静态网站托管
   进入站点详情 → 边缘函数与页面 → 静态网站托管
   开启服务并记录 Zone ID
```

```markdown [获取凭证🔑]
在腾讯云控制台：
1️⃣ 访问「访问管理」→「API 密钥管理」
2️⃣ 创建密钥，获取 SecretId 和 SecretKey
3️⃣ 在 CNB 项目设置中添加环境变量：
   - EO_SECRET_ID: 你的 SecretId
   - EO_SECRET_KEY: 你的 SecretKey
   - EO_ZONE_ID: 你的 Zone ID
```

:::

### 第四步：自动部署

::: code-group

```bash [推送代码触发部署🚀]
# 修改配置后提交
git add .
git commit -m "🎉 初始化博客配置"
git push origin main

# ◀️ CNB 会自动触发构建和部署
# ◀️ 约 2-3 分钟后即可访问
```

```markdown [查看部署状态👀]
1️⃣ 进入 CNB 项目页面
2️⃣ 点击「CI/CD」→「流水线」
3️⃣ 查看构建日志和部署状态
4️⃣ 绿色 ✓ 表示部署成功
```

:::

## 🎯 为什么选择 Hugo-Teek？

### 1️⃣ 真正零成本

::: tip 💡
不需要服务器，不需要数据库，不需要域名（可选），从构建到部署全程免费。
:::

### 2️⃣ 极速体验

::: tip ⚡
Hugo 的构建速度是 Jekyll 的 100 倍，Hexo 的 10 倍。1000 篇文章只需 10 秒构建完成。
:::

### 3️⃣ 数据完全自主

::: tip 🔐
所有内容都是 Markdown 文件，存储在你的 代码 仓库中。随时可以迁移，永远不会丢失。
:::

### 4️⃣ 国内访问快

::: tip 🚀
通过 EO 的国内 CDN 节点，访问速度比 Vercel/Netlify 快 3-5 倍。
:::

### 5️⃣ 专业美观

::: tip 🎨
移植自 VitePress Teek 主题，设计专业，交互流畅，完全不输商业博客平台。
:::
